RotationRandamizer
└エディター拡張本体で、Unity上部"PYEditor"内にあります。


機能説明ーーーーー

指定したオブジェクトの子供すべてに対しローテーションをランダムに設定します。
設定される角度は各オブジェクトで毎回異なります。

☆角度指定
各軸ごとに角度をスナップできます。
0 → 0度に上書き
1 → 1～360度までのランダムな整数値
90 →90度ごとのランダムな整数値
のような具合です。

☆グローバル空間
この拡張ではグローバル空間でRotationを計算します。
親オブジェクトの角度にかかわらず、それぞれのオブジェクトを主体に計算するので思ったように動かないこともあるかもしれません。
